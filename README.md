\# Library Management System



A Full Stack Library Management System built using React and Spring Boot also used tailwind css for styling.



\## Tech Stack

\- Frontend: React (Vite)

\- Backend: Spring Boot

\- Database: PostgreSQL

\- Authentication: JWT



\## Features

\- User Registration \& Login

\- Secure Authentication

\- Add Books

\- Issue Books

\- Return Books

\- View Book List



\## How to Run the Project



\### Backend (Spring Boot)

1\. Go to backend folder

2\. Run:

&nbsp;  mvn spring-boot:run



Server runs on:

http://localhost:8080



\### Frontend (React)

1\. Go to frontend folder

2\. Install dependencies:

&nbsp;  npm install

3\. Run:

&nbsp;  npm run dev



Frontend runs on:

http://localhost:5173
